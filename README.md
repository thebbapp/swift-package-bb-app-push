# ``BbAppPush``

Push notification management and token storage.

## Overview

The BbAppPush library provides push notification infrastructure including device token management and storage. It includes ``BbAppPushTokenStore`` for persisting and retrieving push notification tokens, synchronization with remote servers, and utilities for managing push notification subscriptions. This module coordinates with the app delegate to handle the full push notification lifecycle.
